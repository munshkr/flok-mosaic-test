# flok-mosaic-test

Proof-of-concept for using
[react-mosaic](https://github.com/nomcopter/react-mosaic) to manage **tiling
windows**, for [flok](https://github.com/munshkr/flok).


## Install

Run `yarn` to install dependencies.


## Usage

Start development server:

```
yarn dev
```
